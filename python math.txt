```
python v2.7
3 / 2, 3.0 / 2, 3.0 / 2.0
(1, 1.5, 1.5)

3 / 2, 3.0 / 2, 3 // 2, 3.0 // 2
(1, 1.5, 1, 1.0)
```

===========================
```
v3.x 以后，“/”运算符被命名为“真除”，不再依据操作数类型选择返回值类型，保证计算结果数值上的精度是第一位的。所以，无须再把操作数转变成浮点型，以保证运算结果不被截断小数部分。
“//”运算符，不变。
3 / 2, 3.0 / 2, 3 // 2, 3.0 // 2
(1.5, 1.5, 1, 1.0)
```
